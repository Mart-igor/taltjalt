–û—Ç–ª–∏—á–Ω–æ! –ú–æ–¥—É–ª—å **`re`** –≤ Python –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Ñ—É–Ω–∫—Ü–∏–∏ –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å **—Ä–µ–≥—É–ª—è—Ä–Ω—ã–º–∏ –≤—ã—Ä–∞–∂–µ–Ω–∏—è–º–∏** (regular expressions, regex) ‚Äî –º–æ—â–Ω—ã–º –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–æ–º –¥–ª—è –ø–æ–∏—Å–∫–∞ –∏ –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Ç–µ–∫—Å—Ç–∞.

## üîπ **1. –û—Å–Ω–æ–≤–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –º–æ–¥—É–ª—è `re`**  

### **`re.search(pattern, string)`** ‚Äì –ø–æ–∏—Å–∫ –ø–µ—Ä–≤–æ–≥–æ —Å–æ–≤–ø–∞–¥–µ–Ω–∏—è  
–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç `Match` –æ–±—ä–µ–∫—Ç –∏–ª–∏ `None`, –µ—Å–ª–∏ —Å–æ–≤–ø–∞–¥–µ–Ω–∏–π –Ω–µ—Ç.  
```python
import re

text = "Python 3.10 released in 2021"
match = re.search(r'\d+\.\d+', text)  # –ò—â–µ–º –≤–µ—Ä—Å–∏—é (—á–∏—Å–ª–æ.—á–∏—Å–ª–æ)
if match:
    print(match.group())  # "3.10"
```

### **`re.match(pattern, string)`** ‚Äì –ø—Ä–æ–≤–µ—Ä–∫–∞ —Å–æ–≤–ø–∞–¥–µ–Ω–∏—è **—Å –Ω–∞—á–∞–ª–∞ —Å—Ç—Ä–æ–∫–∏**  
```python
text = "Python is awesome"
match = re.match(r'Python', text)  # –°–æ–≤–ø–∞–¥–µ–Ω–∏–µ –µ—Å—Ç—å
print(match.group())  # "Python"
```

### **`re.findall(pattern, string)`** ‚Äì –ø–æ–∏—Å–∫ **–≤—Å–µ—Ö** —Å–æ–≤–ø–∞–¥–µ–Ω–∏–π  
–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –≤—Å–µ—Ö –Ω–∞–π–¥–µ–Ω–Ω—ã—Ö –ø–æ–¥—Å—Ç—Ä–æ–∫.  
```python
text = "10 apples, 5 bananas, 3 oranges"
numbers = re.findall(r'\d+', text)
print(numbers)  # ['10', '5', '3']
```

### **`re.finditer(pattern, string)`** ‚Äì –∏—Ç–µ—Ä–∞—Ç–æ—Ä —Å `Match` –æ–±—ä–µ–∫—Ç–∞–º–∏  
–ü–æ–ª–µ–∑–Ω–æ –¥–ª—è –æ–±—Ä–∞–±–æ—Ç–∫–∏ –±–æ–ª—å—à–∏—Ö —Ç–µ–∫—Å—Ç–æ–≤.  
```python
text = "Email me at user@example.com or support@site.org"
emails = re.finditer(r'[\w.-]+@[\w.-]+', text)
for email in emails:
    print(email.group())
# user@example.com
# support@site.org
```

### **`re.sub(pattern, repl, string)`** ‚Äì –∑–∞–º–µ–Ω–∞ —Å–æ–≤–ø–∞–¥–µ–Ω–∏–π  
```python
text = "Python 2.7 is old, use Python 3.10"
updated = re.sub(r'2\.7', '3.10', text)
print(updated)  # "Python 3.10 is old, use Python 3.10"
```

### **`re.split(pattern, string)`** ‚Äì —Ä–∞–∑–¥–µ–ª–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫–∏ –ø–æ —à–∞–±–ª–æ–Ω—É  
```python
text = "Apple,Banana;Orange|Grape"
fruits = re.split(r'[,;|]', text)  # –†–∞–∑–¥–µ–ª—è–µ–º –ø–æ , ; –∏–ª–∏ |
print(fruits)  # ['Apple', 'Banana', 'Orange', 'Grape']
```

## üîπ **2. –°–∏–Ω—Ç–∞–∫—Å–∏—Å —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏–π**  

### **–ë–∞–∑–æ–≤—ã–µ –º–µ—Ç–∞—Å–∏–º–≤–æ–ª—ã**  
| –ü–∞—Ç—Ç–µ—Ä–Ω  | –û–ø–∏—Å–∞–Ω–∏–µ                     | –ü—Ä–∏–º–µ—Ä                     |
|----------|------------------------------|----------------------------|
| `.`      | –õ—é–±–æ–π —Å–∏–º–≤–æ–ª (–∫—Ä–æ–º–µ `\n`)    | `a.c` ‚Üí "abc", "a c"       |
| `\d`     | –¶–∏—Ñ—Ä–∞ (`[0-9]`)              | `\d+` ‚Üí "123"              |
| `\D`     | –ù–µ —Ü–∏—Ñ—Ä–∞ (`[^0-9]`)          | `\D+` ‚Üí "abc"              |
| `\w`     | –ë—É–∫–≤–∞, —Ü–∏—Ñ—Ä–∞ –∏–ª–∏ `_`         | `\w+` ‚Üí "user123"          |
| `\W`     | –ù–µ `\w`                      | `\W+` ‚Üí "@#$"              |
| `\s`     | –ü—Ä–æ–±–µ–ª—å–Ω—ã–π —Å–∏–º–≤–æ–ª            | `\s+` ‚Üí " \t\n"            |
| `\S`     | –ù–µ –ø—Ä–æ–±–µ–ª—å–Ω—ã–π —Å–∏–º–≤–æ–ª         | `\S+` ‚Üí "abc"              |

### **–ö–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã (–ø–æ–≤—Ç–æ—Ä—ã)**  
| –ü–∞—Ç—Ç–µ—Ä–Ω  | –û–ø–∏—Å–∞–Ω–∏–µ                     | –ü—Ä–∏–º–µ—Ä                     |
|----------|------------------------------|----------------------------|
| `*`      | 0 –∏–ª–∏ –±–æ–ª—å—à–µ                 | `a*` ‚Üí "", "a", "aa"       |
| `+`      | 1 –∏–ª–∏ –±–æ–ª—å—à–µ                 | `\d+` ‚Üí "1", "123"         |
| `?`      | 0 –∏–ª–∏ 1                      | `a?b` ‚Üí "b", "ab"          |
| `{n}`    | –†–æ–≤–Ω–æ `n` —Ä–∞–∑                | `a{3}` ‚Üí "aaa"             |
| `{n,}`   | `n` –∏–ª–∏ –±–æ–ª—å—à–µ               | `\d{2,}` ‚Üí "12", "123"     |
| `{n,m}`  | –û—Ç `n` –¥–æ `m` —Ä–∞–∑            | `a{2,4}` ‚Üí "aa", "aaa"     |

### **–ì—Ä—É–ø–ø—ã –∏ —É—Å–ª–æ–≤–∏—è**  
| –ü–∞—Ç—Ç–µ—Ä–Ω         | –û–ø–∏—Å–∞–Ω–∏–µ                     | –ü—Ä–∏–º–µ—Ä                     |
|-----------------|------------------------------|----------------------------|
| `(...)`         | –ì—Ä—É–ø–ø–∞                       | `(ab)+` ‚Üí "abab"           |
| `(?:...)`       | –ù–µ–∑–∞—Ö–≤–∞—Ç—ã–≤–∞—é—â–∞—è –≥—Ä—É–ø–ø–∞       | `(?:ab)+` ‚Üí "abab"         |
| `\|`            | –ò–õ–ò                          | `cat\|dog` ‚Üí "cat", "dog"  |
| `^`             | –ù–∞—á–∞–ª–æ —Å—Ç—Ä–æ–∫–∏                | `^Hello` ‚Üí "Hello world"   |
| `$`             | –ö–æ–Ω–µ—Ü —Å—Ç—Ä–æ–∫–∏                 | `end$` ‚Üí "The end"         |

## üîπ **3. –§–ª–∞–≥–∏ (–º–æ–¥–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã)**  
–§–ª–∞–≥–∏ –∏–∑–º–µ–Ω—è—é—Ç –ø–æ–≤–µ–¥–µ–Ω–∏–µ —Ä–µ–≥—É–ª—è—Ä–Ω–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è:  

| –§–ª–∞–≥            | –û–ø–∏—Å–∞–Ω–∏–µ                     | –ü—Ä–∏–º–µ—Ä                     |
|-----------------|------------------------------|----------------------------|
| `re.IGNORECASE` | –ò–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞—Ç—å —Ä–µ–≥–∏—Å—Ç—Ä         | `re.search(r'python', 'PYTHON', re.I)` |
| `re.MULTILINE`  | `^` –∏ `$` –¥–ª—è –∫–∞–∂–¥–æ–π —Å—Ç—Ä–æ–∫–∏  | `re.findall(r'^\d+', '1\n2\n3', re.M)` ‚Üí ["1", "2", "3"] |
| `re.DOTALL`     | `.` –≤–∫–ª—é—á–∞–µ—Ç `\n`            | `re.search(r'a.b', 'a\nb', re.S)` |

–ü—Ä–∏–º–µ—Ä:  
```python
text = "Python\npython\nPYTHON"
matches = re.findall(r'python', text, re.IGNORECASE)
print(matches)  # ['Python', 'python', 'PYTHON']
```
