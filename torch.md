### **5. –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏–µ (Autograd)**  

#### **TensorFlow (GradientTape)**  
```python
with tf.GradientTape() as tape:
    y = x ** 2
dy_dx = tape.gradient(y, x)  # dy/dx = 2x
```

#### **PyTorch (Autograd)**  
```python
x = torch.tensor(2.0, requires_grad=True)
y = x ** 2
y.backward()
dy_dx = x.grad  # dy/dx = 2x ‚Üí 4.0
```
